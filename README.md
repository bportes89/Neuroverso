# Neuroverso

## Objectives
- Provide an immersive experience for users in a virtual environment.
- Enable user interactions through multiple functionalities.

## MVP Scope
| Feature               | Description                    |
|-----------------------|--------------------------------|
| User Authentication    | Users can sign up and log in. |
| Virtual Environment     | Users navigate a 3D space.     |
| Interactive Elements    | Engage with objects in the environment. |

## Simplified Architecture
- Client-Server Model
- Frontend: React
- Backend: Node.js with Express

## Main Flows
- **User Login**: User enters credentials, accesses virtual environment.
- **Navigation**: Users can explore different areas seamlessly.
- **Interaction**: Users engage with elements in the environment for further actions.

## Technology Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Hosting**: AWS

## Quick Start Setup
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the server: `npm start`
